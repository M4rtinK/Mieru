Mieru - manual package installation on Harmattan

NOTE: installation from Ovi store or Apps for MeeGo doesn't need developer mode and is fully automatic

  1. become root by running 

    devel-su
    with developer mode enabled

  2. manually install all dependencies needed by Mieru 
    apt-get update
apt-get install libzip1, libarchive1, python-pyside.qtgui, python-pyside.qtdeclarative, python-magic, python
  3. download the Mieru installation package from the Mieru OBS repository
  4. install the package
    dpkg -i CURRENT_MIERU_PACKAGE_NAME
